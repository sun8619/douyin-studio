# 抖音直播间设计参考 - 设计规范文档

> 基于 24 张真实抖音直播间截图分析整理

---

## 一、整体布局（竖屏 9:16）

**三段式结构（上下分区）：**
- **顶部区（Top 15%）**：品牌背书 + 活动主题，典型元素：机构名称、专场标签
- **中部区（Middle 55%）**：主播 + 核心商品/优惠券，视觉焦点区
- **底部区（Bottom 30%）**：品牌文案 + 画中画（菜品小图）+ 互动区

**重要设计规律：**
- 上中下区用"撕纸边缘"效果分隔，打破单调感
- 信息密度高但逻辑分层清晰
- 竖屏比例（9:16），适配手机观看习惯

---

## 二、背景设计

**主流风格（4种）：**

### 1. 复古中式餐饮风（最常见）
- 做旧报纸纹理 / 砖墙纹理
- 红色灯笼 + 福字 + 折扇
- 乡土道具：铁锅、木材、玉米、土灶
- 色彩：红 + 米色/卡其色

### 2. 纯色 + 品牌元素
- 纯色渐变背景（品牌色）
- 产品图大字报风格
- 简洁促销型，无复杂背景

### 3. 真实场景型
- 厨房/门店真实拍摄背景
- 主播在实际经营场所直播
- 强调"真实感"和"烟火气"

### 4. 创意舞台型
- 幕布 + 聚光灯
- 展板/LED 大屏背景
- 品牌 VI 色为主色调

## 三、UI 元素位置（标准化）

```
┌──────────────────────────────────┐
│ 🏠 仓库  ♡点赞  👥在线人数  │ ← 顶部信息栏（半透明）
├──────────────────────────────────┤
│ [机构背书｜专场名称]            │ ← 顶部 banner
│                                  │
│  ┌────────┐                     │
│  │ 商品卡片│ ← 左上悬浮        │
│  │ 名称+价格│                   │
│  └────────┘                     │
│                                  │
│        【主播区域】              │ ← 中央偏下
│     + 手持商品/优惠券            │
│                                  │
│  ┌─促销标签─┐   ┌─优惠券─┐     │ ← 右侧/中右
│  │ 769代1000│   │299代500│     │
│  └──────────┘   └────────┘     │
│                                  │
│  ┌────────────────────────────┐ │
│  │ 📢 弹幕互动区（飘过动画）   │ │ ← 底部弹幕
│  └────────────────────────────┘ │
│  [🍜 品牌文案]  [小图预览]      │ ← 底部
└──────────────────────────────────┘
```

**标准化 UI 组件：**
1. **商品卡**：左上悬浮，白底带阴影，品名+价格+购买按钮
2. **促销优惠券**：右/中右，大字红底白字，"代"字突出，"限时"标签
3. **关注引导**：左下角，"点关注不迷路"类话术
4. **弹幕区**：底部飘过动画，半透明底色
5. **在线数据**：右上角，真实感数字跳动

## 四、色彩体系

**主流配色（按行业）：**

| 行业 | 主色 | 辅色 | 背景色 |
|------|------|------|--------|
| 火锅/烤肉 | 正红 #FF2020 | 金色 #FFD700 | 深红/黑色 |
| 奶茶/烘焙 | 马卡龙粉/薄荷绿 | 白色 | 浅粉/浅绿 |
| 海鲜 | 深海蓝 #1A5F7A | 橙红 | 深蓝/黑色 |
| 小龙虾 | 麻辣红 | 金色 | 深红黑 |
| 面馆/快餐 | 橙黄 | 白色 | 米白/浅棕 |
| 景区/文旅 | 山绿/天蓝 | 土棕 | 山水背景 |

**通用规律：**
- 背景深色系（便于内容突出）
- 促销元素高饱和红色（紧迫感、刺激购买）
- 辅色克制（1-2色），主色突出

## 五、物料类型（我们的产品要生成的）

1. **直播封面图**：开场用，大字报风格，突出主题+优惠
2. **优惠卡片**：右/中右悬浮，"代"字系（299代500等）
3. **弹幕提示卡**：底部互动话术，自动生成
4. **关注引导图**：左下角引导关注
5. **商品展示卡**：左上悬浮商品信息卡
6. **促销标签**：顶部/右侧醒目促销信息

## 六、设计细节规范

**撕纸边缘效果：**
- 上、下分区边缘使用不规则"撕开"边缘
- 颜色：白底 + 浅灰阴影，制造立体感

**投影与层次：**
- 卡片：box-shadow: 0 4px 20px rgba(0,0,0,0.3)
- 悬浮层叠，多个卡片时z-index区分

**字体规范：**
- 标题/价格：加粗黑体 / Impact 风格
- 优惠数字：超大字号（2-3倍正文）
- 装饰元素：楷体/宋体（古风场景）

**动画效果：**
- 弹幕：从右向左飘过，速度适中
- 点赞数字：跳动更新
- 优惠卡片：轻微 pulse 效果吸引注意

## 七、「抖店内容工坊」直播间模拟页面设计建议

**左侧直播间模拟（参考真实抖音布局）：**
1. 顶部信息栏：仓库+点赞+在线人数（半透明黑底）
2. 顶部 banner：支持切换主题，显示对应品牌信息
3. 中央区域：
   - 纯色/渐变背景（随主题变化）
   - 主播占位区（中间偏下）
   - 可叠加物料预览（优惠券/商品卡实时显示）
4. 弹幕区：底部，实时飘过弹幕动画
5. 右下角小窗：当前物料预览

**右侧控制台：**
- 主题切换 → 联动左侧背景/色调变化
- 物料类型切换 → 实时预览该物料叠加效果
- 生成后 → 物料直接出现在左侧画面中

**风格要求：**
- 深色主题，参考抖音直播伴侣
- 顶部信息栏半透明，不遮挡背景
- 物料悬浮位置参考真实直播间比例
