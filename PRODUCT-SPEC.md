# 抖店内容工坊 - 产品规格书

> 版本：v0.1 MVP  
> 最后更新：2026-05-13  
> 状态：进行中

---

## 一、产品定位

**一句话定位**：商家输入一个营销主题，平台智能生成文案+图片+视频+直播物料的全链路内容创作工具。

**差异化价值**：
- 纵向联动：脚本 → 用户修改确认 → 直接生成对应短视频
- 横向联动：同一营销主题 → 批量产出多形式内容（文案+图片+视频+直播物料）
- 内容自带热榜加速度（接入热点情报）

---

## 二、目标用户

| 维度 | 描述 |
|------|------|
| 用户画像 | 餐饮、酒旅、休闲娱乐等抖音生活服务中小商家 |
| 核心痛点 | 有内容需求但缺乏专业团队，不知道发什么、怎么发 |
| 使用场景 | 日常运营、节日促销、新品上市、直播预热 |
| 付费意愿 | 基础文案免费，图片/视频/直播物料付费会员 |

---

## 三、功能模块

### 3.1 文案创作（免费基础功能）

**子功能**：
- 📝 **脚本生成**：商家输入行业+品类+主题，生成短视频口播脚本
- 💬 **直播话术**：生成直播开场/促单/互动话术
- 📖 **种草文**：生成小红书/抖音图文种草文案
- 💬 **评论回复**：输入顾客评论，生成回复建议

**行业覆盖**：火锅、烧烤、奶茶、烘焙、酒店、民宿、健身房、美容院等

**横向联动**：文案 → 自动推荐匹配的图片风格 + 视频脚本

---

### 3.2 图片设计（付费会员）

**子功能**：
- 🖼️ **商品套图**：菜品/产品多角度展示图
- 🎨 **营销海报**：节日/促销海报
- 🏠 **店铺装修图**：门头、菜单封面、导视系统
- 📺 **直播背景**：直播间背景板、贴片设计

**提示词支持**：内置行业模板化提示词库，降低商家使用门槛

---

### 3.3 短视频（付费会员）

**子功能**：
- 脚本 → 用户确认 → AI 生成视频（接入即梦/可灵 API）
- 各账号类型独立模块：店铺官方账号 / 老板账号 / 职人账号 / 达人账号
- 视频生成后：标题 + 封面 + 标签建议（video-optimization 提供）

---

### 3.4 直播物料（付费会员）

**子功能**：
- 直播间背景设计
- 贴片设计（右下角互动贴片）
- 预热视频生成
- 直播脚本文案

---

## 四、内容联动机制

### 纵向联动
```
营销主题 → 文案生成（脚本/话术）→ 用户修改确认 → 直接生成对应短视频
```

### 横向联动
```
营销主题
    ├── 推广文案（种草文/评论回复）
    ├── 商品主图 + 场景图
    ├── 短视频脚本
    ├── 直播间背景设计
    └── 直播预热视频
```

## 五、账号体系

| 账号类型 | 定位 | 内容特点 |
|----------|------|----------|
| 店铺官方账号 | 品牌门面 | 正式、官方、权威 |
| 老板账号 | 创始人IP | 个性化、真人出镜 |
| 职人账号 | 店员/员工 | 日常运营、接地气 |
| 达人账号 | KOL/合作达人 | 专业、有影响力 |

每个账号模块独立设置生成流。
